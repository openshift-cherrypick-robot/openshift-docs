= Connecting Azure AD to RHACS using SSO configuration
include::modules/common-attributes.adoc[]
:context: user-authentication

toc::[]

[role="_abstract"]

To connect an Azure Active Directory (AD) to {product-title-short} using Sign-On (SSO) configuration, you need to add specific claims (for example, `group` claim to tokens) and assign users, groups, or both to the enterprise application.

[id="adding-group-claims-to-tokens-for-saml-applications-using-SSO-configuration_{context}"]
== Adding group claims to tokens for SAML applications using SSO configuration

Configure the application registration in Azure AD to include `group` claims in tokens. For instructions, see link:https://learn.microsoft.com/en-us/azure/active-directory/hybrid/how-to-connect-fed-group-claims#add-group-claims-to-tokens-for-saml-applications-using-sso-configuration[Add group claims to tokens for SAML applications using SSO configuration].

[IMPORTANT]
====
Verify that you are using the latest version of Azure AD.
For more information on how to upgrade Azure AD to the latest version, see link:https://learn.microsoft.com/en-us/azure/active-directory/hybrid/how-to-upgrade-previous-version[Azure AD Connect: Upgrade from a previous version to the latest].
====
